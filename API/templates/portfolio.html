Création d'une application en Python

Heures valorisées : 10h

Fin 2020, j'ai commencé la conception d'une application en python. Cette application a pour but de transformer une partition de musique en un système plus simple à la lecture
Cette application possède une page d'accueil contenant deux bouton permettant de se rendre au piano virtuel (où des notes peuvent être placées) et vers une page de capture d'image. Une fois la photo réalisée elle est retranscrite en code correspondant aux notes de musique, clef et longueur de note. Cela est réalisé par une système de Machine Learning trouvé sur internet, ce système est monophonic, c-à-d qu'il ne peut lire que des partitions avec une seule mélodie (voix)
Fin 2020
